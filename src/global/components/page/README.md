Page
=====

Page is the component that wraps all elements of each individual 'page'
******
Props
=====
| Prop          | Type    | Description                |
| ------------- |-------| -----                    |
| classes       | String  | Custom class name         |

******
Example
=======

    import Page from 'global/components/page';

    const ExamplePage = () => {
        return (
            <Page classes = 'example-Page'>
                Example Text
            </Page>
        )
    }
